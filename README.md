# ShopVoiceLog AI

Enhanced grocery assistant with:
- Voice-driven product logging
- Smart suggestions
- Video shelf-scanning
- Gemma agentic reasoning
- Real retailer integration

## Structure
- frontend/   → React UI
- backend/    → Node API
- ai/gemma/   → Gemma reasoning
- ai/vision/  → Vision + OCR
- ai/agent/   → Multi-step orchestration
- docs/       → Architecture & specs
- infra/      → Deployment & ops
- scripts/    → Automation

## Getting Started
1. cd backend && npm run dev
2. cd frontend && npm run dev
3. Implement Gemma, vision, and agent logic in ai/ folders.
