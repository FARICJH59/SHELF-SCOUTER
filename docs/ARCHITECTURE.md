# ShopVoiceLog AI – Architecture

Layers:
1. Frontend (React + Vite + TS)
2. Backend API (Node + Express + TS)
3. AI Layer
   - Gemma (agentic reasoning, intent understanding)
   - Vision (YOLO/vision + OCR for shelf scouting)
   - Agent (multi-step orchestration)
4. Retailer Integrations (catalogs, prices, SKUs)
5. User Experience (voice, video shelf scouting, smart suggestions)
